projectDOAMA API

projectDOAMA allows API calls to be made against our ever-growing databse of movies, actors, and genres.

Movie Information

To recieve the full database of movies, search:

http://www.projectdoama.com/api/v1/getMovieInfo

To recieve information about a specific movie, search:

http://www.projectdoama.com/api/v1/getMovieInfo/[[INSERT MOVIE TITLE HERE]]

Genre Information

To recieve the full database of genres, search:

http://www.projectdoama.com/api/v2/getGenreInfo/[[INSERT GENRE HERE]]

Actor Information

To recieve the full database of actors, search:

http://www.projectdoama.com/api/v3/getActorInfo/[[INSERT ACTOR HERE]]
